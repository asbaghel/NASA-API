# NASA API

This is project based on ReactJS it uses API provided by nasa to fetch data.I have used Material UI for styling.


## Requirements 
* ReactJs


## Getting started

 1. Run `npm install` , all required dependencies will be installed automatically.
 2. Run `npm start`
